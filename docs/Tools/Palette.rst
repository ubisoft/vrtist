Palette
=======

.. Palette:
.. include:: /Refs.rst

| The palette contains all the tools available to the user. To open it, use the trigger on the primary controller.
| Some options are available at the top of the palette. 
| 
| Click the |HomeIcon| icon to return to the lobby.
|
| Click the |SaveIcon| icon to open the save and load setting panel.
|
| Click the |PlayIcon| icon, to play scene animations.
|
| Click the |ShowIcon| icon to show/hide gizmos in the scene.
|
| Click the |AxisIcon| icon to show/hide locators in the scene.

| The palette can be pinned around the user. To do so, click on the |PinIcon| icon, and move the palette to a new position.
| The palette position is saved between sessions.
|
